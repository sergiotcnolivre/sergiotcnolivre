## Sérgio Campos :brazil:


Com anos de experiência em desenvolvimento de software, atualmente contribuo como Desenvolvedor Java. 
Nosso time está transformando sistemas bancários críticos no projeto Global Account, utilizando tecnologias como Java, Kafka, AWS, SQS, SNS e ferramentas de observabilidade como New Relic. 
Nosso objetivo é aprimorar a eficiência e escalabilidade do Core Banking, promovendo inovação no setor financeiro.  

Meu trabalho reflete um compromisso com a modernização tecnológica e a otimização de processos essenciais. Além disso, anteriormente atuei como professor na Arnia, onde capacitei novos talentos em programação web, promovendo boas práticas de desenvolvimento e versionamento. Acredito no poder da tecnologia para criar soluções impactantes e moldar o futuro do setor.
